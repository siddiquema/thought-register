# Thought Register

**Capture first. Understand later.**

An open-source, offline-first knowledge capture system for recording thoughts, observations, experiences, questions and insights before they are forgotten.

## Vision
Capture ideas with less than 30 seconds of effort.

## Version 0.1
- Quick Capture
- Structured Capture
- Local Storage
- Markdown Export
- CSV Export
- Search
- Installable PWA
