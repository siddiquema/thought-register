# Vision

Build the world's simplest open-source knowledge capture system.

Not a note-taking application. Not a PKM platform. Not another Obsidian.

A frictionless tool for capturing observations, ideas, experiences and insights before they disappear.

## What "done" looks like

Someone has a thought. They open Thought Register, write it down, and it's saved — in under 30 seconds, on any device, with or without a network connection. They can search it later, export it in a format they own, and never have to wonder whether the tool will still exist, or still let them leave, in five years.

## What this project optimizes for

- **Speed of capture** over richness of organization.
- **Ownership** over convenience — the user's data, in a plain format, on their device, always exportable.
- **Longevity** over trend-following — HTML, CSS and vanilla JavaScript outlast frameworks.
- **Trust** over engagement — no analytics, no telemetry, nothing trying to keep anyone using it longer than they need to.
