# Manifesto

People lose valuable ideas because capturing them takes too much effort.

By the time a note-taking app has loaded, asked which notebook, which tags, which folder — the thought is gone. Thought Register exists to preserve human thinking with the least possible friction, so the only thing standing between a person and their idea is the idea itself.

Capture first. Understand later.

Organizing, tagging, linking and reflecting are valuable — but they are not the first problem. The first problem is that the thought escapes before any of that can happen. Thought Register solves the first problem, deliberately, and leaves the rest to the person who owns the thought.

If a feature slows down capturing, it does not belong here — no matter how useful it seems afterward.

## Principles

- Simplicity over Features
- Offline First
- Mobile First
- Markdown First
- Privacy First
- Open Source
- Human Owned
- AI Optional
- Export Everything
- No Vendor Lock-in
- Progressive Enhancement
- Accessibility
- Fast
- Lightweight

## The Test

Every decision — a feature, a dependency, a screen, a setting — is measured against the same question:

**Does this help someone capture knowledge faster?**

If yes, it may belong. If no, it doesn't — regardless of how interesting it is.
