# Thought Register

**Capture first. Understand later.**

Thought Register is an open-source, offline-first knowledge capture system. It exists to record thoughts, observations, experiences, questions and insights before they disappear — not to organize, plan, or manage them.

It is not a note-taking app, a PKM platform, or another Obsidian. See [NON_GOALS.md](NON_GOALS.md) for what this project deliberately avoids becoming.

## Why

People lose valuable ideas because capturing them takes too much effort. Every design decision in this project is measured against one metric: the time from opening the app to saving a thought. Read the full reasoning in [MANIFESTO.md](MANIFESTO.md) and [VISION.md](VISION.md).

## Success Metric

**Under 30 seconds** from open to save.

## Version 0.1 Scope

- Quick Capture
- Structured Capture
- Local Storage
- Markdown Export
- CSV Export
- Search
- Dark Mode
- Installable PWA

Nothing more. See [ROADMAP.md](ROADMAP.md) for what comes after, and [NON_GOALS.md](NON_GOALS.md) for what never will.

## Principles

Simplicity over features · Offline First · Mobile First · Markdown First · Privacy First · Open Source · Human Owned · AI Optional · Export Everything · No Vendor Lock-in · Progressive Enhancement · Accessibility · Fast · Lightweight

## Technology

HTML, CSS, and vanilla JavaScript. No frameworks, no backend, no database, no accounts, no cloud dependency, no analytics, no telemetry. Dependency-free wherever possible.
